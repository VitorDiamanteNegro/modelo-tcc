Sobre o FlashStudy

O FlashStudy foi desenvolvido como projeto de TCC para revolucionar a maneira como os estudantes utilizam flashcards para aprendizado.

Utilizamos técnicas modernas de repetição espaçada e design intuitivo para maximizar a eficiência do seu estudo.
